Semantic UI

Semantic UI är ett ramverk med öppen källkod för att bygga intuitiva användargränssnitt med CSS, jQuery och HTML. Semantic UI fungerar som en överlagring ovanpå React-komponenterna och tillhandahåller semantiska teman som CSS-stilmall. Detta front-end-utvecklingsramverk hjälper utvecklare att skapa webbplatser med snabb och koncis HTML i kombination med en komplett mobil-responsiv upplevelse. Semantic UI erbjuder många komponenter för gränssnittsdesign och är ett jQuery-fritt, React-kompatibelt ramverk som Bootstrap-React.

Fördelar med Semantic UI:

  • Verkar väldigt enkelt att använda med tydlig semantik
  • Finns ikoner och snygga knappar och kompbonenter som är lätt att implementera

Nackdelar med Semantic UI:

  • verkar finnas färre responsiva element. Ex, saknar responsiv meny?
  • Inte lika många användare, vilket gör det svårare att felsöka på forum
